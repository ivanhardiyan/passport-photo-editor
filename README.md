# Passport Photo Editor

## Description
The Passport Photo Editor application is designed to help users create professional-looking passport photos easily. Users can upload their images, apply necessary adjustments, and download high-quality photos ready for printing. 

## Features
- **Image Upload**: Users can upload pictures directly from their devices  
- **Background Removal**: Automatic background removal to ensure compliance with passport photo standards  
- **Cropping & Resizing**: Easy tools to crop and resize images to meet specific passport photo dimensions  
- **Filters & Adjustments**: Apply filters and adjust brightness, contrast, and saturation for optimal results  
- **Download Options**: Export images in various formats suitable for printing  

## Tech Stack
- **Frontend**: React.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Hosting**: AWS

## Setup Instructions
1. Clone the repository: `git clone https://github.com/ivanhardiyan/passport-photo-editor`
2. Navigate into the project directory: `cd passport-photo-editor`
3. Install dependencies: `npm install`
4. Start the server: `npm start`
5. Visit `http://localhost:3000` in your browser.

## Usage Guide
1. Upload your photo by clicking on the upload button.
2. Use the editing tools to adjust your photo as needed.
3. Click on the download button to save your passport photo once you're satisfied with the edits.

For further assistance, consult the documentation or raise an issue on the GitHub repository.
